1. cd to the RUNscripts directory. Edit the RUNTurkey.bat file in any text editor and change the path to the directory where this README is.

2. Click on the RUNTurkey script to launch Wiggleview.

3. Basic keys:
F to toggle fullscreen
F4 to switch between globe and flat map
WASD to rotate
QE to rotate along the z-axis
Enter to toggle spin
Alt+P to play at highest speed
T to toggle text
UO to zoom in/out
IJKL to pan

See http://www.evl.uic.edu/cavern/agave/wiggleview for detailed documentation.
